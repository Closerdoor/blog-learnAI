## 关于本项目

这是一个记录 AI 学习路线规划、资料筛选和项目实践的 GitBook 仓库。

当前目标不是单纯“会用 AI”，而是逐步掌握 **AI 应用开发、RAG、Tool Calling、Agent Workflow、工程化部署**，形成可用于求职和长期积累的全栈 AI 工程能力。

## 学习目标

- 建立面向企业场景的 AI / Agent 工程能力
- 使用 Python 和 TypeScript 构建可落地的 AI 应用
- 掌握 RAG、工具调用、工作流和 Agent 状态管理
- 沉淀可展示、可复盘、可面试讲解的项目作品

## 技术背景

- 前端开发经验 (TypeScript/JavaScript)
- 当前工作中大量使用 Python
- 对 WASM/Rust 有兴趣
