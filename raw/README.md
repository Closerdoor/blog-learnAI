# Raw Notes

> 零散笔记存放目录。尚未归类的想法、代码片段、资料评价和问题先放这里，再定期整理到 `planning/` 或 `docs/`。

## 使用方式

1. 在此目录下创建任意 `.md` 文件
2. 随意记录学习内容、代码、问题等
3. 定期让 AI 帮你整理到 `docs/` 对应章节

## 文件命名建议

- `prompt-notes.md` - Prompt 相关笔记
- `rag-notes.md` - RAG 相关笔记
- `agent-notes.md` - Agent 相关笔记
- `resource-review.md` - 资料评估记录
- 或任意你喜欢的名称
