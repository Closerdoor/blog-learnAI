# 第二阶段：RAG 与知识库构建

> 目标：构建能处理私有资料的知识库系统，为后续 Agent 提供可靠上下文。

## 核心概念

- Embedding (向量化)
- Chunking (分块策略)
- Vector Database (向量数据库)
- Hybrid Search
- Rerank
- RAG 评估

## 技术栈

- LlamaIndex (处理复杂结构化数据)
- Chroma / Supabase Vector / Qdrant
- OpenAI Embeddings

## 实操项目

- CodePilot Lab：代码库理解和上下文检索
- Personal Media Graph：个人收藏内容的语义检索和推荐
