## 面试资料

AIGC-Interview-Book 作为查漏补缺的字典，重点章节：
- `08. RAG` - 向量数据库、检索优化
- `10. Agent` - ReAct 框架、Tool Calling
- `12. 模型推理与部署` - 量化、端侧推理
- `03. 大模型基础知识` - Token、Temperature、Context Window

agent-study 作为 Agent 知识地图和面试补充，重点章节：
- `Ch1` - 第一个 Agent，ReAct 循环和 Function Calling
- `Ch4` - LangChain / LangGraph 框架概览
- `Ch9` - RAG 技术演进
- `Ch11` - Tool Calling 底层机制
- `Ch19` - Agentic Workflow 设计模式
- `Ch20` - Context Engineering
