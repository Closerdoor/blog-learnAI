# Prompt Engineering

> 目标：掌握让模型稳定完成任务的基础表达方式，并为后续 Structured Output、Tool Calling 和 Agent Workflow 打基础。

## 学习重点

- 指令、上下文、示例、约束和输出格式
- 任务拆解：总结、分类、抽取、改写、判断
- Few-shot 示例设计
- 输出稳定性和可验证性
- Prompt 与工具描述的关系

## 项目连接

- AI 学习资料整理助手：资料分类、推荐理由、GitBook 草稿生成
- GitHub 仓库评估器：仓库价值判断和结构化报告生成
