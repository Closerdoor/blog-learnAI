# Prompt Engineering

> 目标：掌握让模型稳定完成任务的基础表达方式，并为后续 Structured Output、Tool Calling 和 Agent Workflow 打基础。

## 学习重点

- 指令、上下文、示例、约束和输出格式
- 任务拆解：总结、分类、抽取、改写、判断
- Few-shot 示例设计
- 输出稳定性和可验证性
- Prompt 与工具描述的关系

## 项目连接

- CodePilot Lab：任务理解、修改计划、执行总结
- Agent Tool Platform：用户指令解析、工具选择说明
- Personal Media Graph：内容分类、推荐理由生成
