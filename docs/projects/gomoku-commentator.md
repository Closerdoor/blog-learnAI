# 五子棋解说 Agent

> 目标：把兴趣项目升级成能体现 Tool Calling、状态管理和流式交互的 Agent 项目。

## 项目定位

这是 Agent 阶段的兴趣项目，不作为最早的主线项目。

## 核心功能

- 读取棋局状态
- 调用分析工具
- 判断局势变化
- 实时生成解说
- 展示胜率或局势评价
- 支持不同解说风格

## 技术栈

- Python
- LangGraph
- Tool Calling
- TypeScript / Streaming UI

## 学习目标

- 练习 Agent 状态管理
- 练习工具调用和结果解释
- 练习流式输出和前端展示
