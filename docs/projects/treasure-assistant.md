# 资料馆智能助手

> 目标：把个人资料、收藏、Markdown、链接等内容变成可检索、可问答、可推荐的知识库。

## 项目定位

这是 RAG 阶段的核心项目，也可以逐步升级成 Agent 项目。

## 核心功能

- 文档导入
- 自动分类
- 语义搜索
- 智能推荐
- 带引用的问答
- 资料质量评估

## 技术栈

- Python
- LlamaIndex
- Embeddings
- Vector Database
- FastAPI

## 学习目标

- 掌握 RAG 工程基础
- 理解 Chunking、Embedding、检索和引用
- 积累可展示的 AI 应用项目
