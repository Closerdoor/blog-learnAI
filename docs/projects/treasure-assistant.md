# Personal Media Graph

> 目标：把个人收藏馆升级为围绕电影、动漫、游戏、音乐、书籍的知识图谱、语义检索和推荐系统。

## 项目定位

这是长期兴趣产品，不作为第一求职主线。

它用于训练：

- RAG
- Knowledge Graph
- 语义搜索
- 个性化推荐
- 多源资料管理
- 长期记忆

## 核心功能

- 作品收藏：电影、动漫、游戏、音乐、书籍
- 元数据管理：人物、公司、系列、标签、时间线
- 评论和个人笔记管理
- 作品关系图谱
- 相似作品推荐
- 自然语言搜索
- 推荐理由解释

## 技术栈

- Python
- RAG
- Knowledge Graph
- Vector Database
- Recommendation

## 学习目标

- 掌握 RAG 和图谱结合的方式
- 理解个人长期记忆和推荐系统
- 保持与个人兴趣绑定的长期产品动力
