# 实战项目

> 边学边做，用真实项目驱动学习

当前项目分为三类：

- **主线求职作品**：CodePilot Lab
- **强相关副作品**：Agent Tool Platform
- **长期兴趣产品**：Personal Media Graph
- **兴趣实验**：Web 游戏站点、五子棋等

## 项目列表

### 1. CodePilot Lab

- **技术栈**: Python + LangGraph + Tool Calling + Git + Shell + Test Runner
- **功能**: 读取代码库、制定修改计划、编辑文件、运行测试、生成 diff、记录执行轨迹
- **阶段**: 第二至第五阶段
- **定位**: 主线求职作品，面向 Agentic Coding

### 2. Agent Tool Platform

- **技术栈**: Python + Tool Registry + Plugin System + Permission + Audit Log
- **功能**: 多通道消息接入、AI 指令路由、插件调用、权限控制、限流和审计
- **阶段**: 第三至第五阶段
- **定位**: 由 QQ 机器人兴趣升级而来的第二作品

### 3. Personal Media Graph

- **技术栈**: Python + RAG + Knowledge Graph + Recommendation
- **功能**: 管理电影、动漫、游戏、音乐、书籍、评论和笔记，构建语义检索和推荐系统
- **阶段**: 第二阶段起长期积累
- **定位**: 长期兴趣产品，训练 RAG、知识图谱、推荐和长期记忆

### 4. Web 游戏站点 / 五子棋实验

- **技术栈**: TypeScript + Web Game + Playwright
- **功能**: 复古小游戏集合，五子棋 AI 和先手必胜研究
- **阶段**: 兴趣实验
- **定位**: 暂不纳入职业主线

### 5. 学习辅助工具

- **技术栈**: Python + OpenAI API + RAG + Markdown
- **功能**: GitBook 资料问答、资料整理、学习路线规划
- **阶段**: 按需
- **定位**: 服务学习过程，不作为主要求职作品
