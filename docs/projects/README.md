# 实战项目

> 边学边做，用真实项目驱动学习

## 项目列表

### 1. AI 学习资料整理助手

- **技术栈**: Python + OpenAI API + Structured Output + Markdown
- **功能**: 对学习资料进行分类、推荐、阶段匹配，并生成 GitBook 草稿
- **阶段**: 第一阶段

### 2. GitBook 资料问答助手

- **技术栈**: RAG + LlamaIndex + Vector Database
- **功能**: 读取 GitBook 笔记，支持资料检索、问答和引用来源
- **阶段**: 第二阶段

### 3. GitHub 仓库评估器

- **技术栈**: Python + GitHub API + Tool Calling + FastAPI
- **功能**: 分析仓库 README、目录和示例代码，判断是否纳入学习主线
- **阶段**: 第三阶段

### 4. 学习路线规划 Agent

- **技术栈**: LangGraph / OpenAI Agents SDK + RAG + Human-in-the-loop
- **功能**: 根据目标、时间和资料库生成学习计划，并记录决策过程
- **阶段**: 第四阶段

### 5. 五子棋解说 Agent

- **技术栈**: LangGraph + Tool Calling + Streaming UI
- **功能**: 分析棋局、实时解说、胜率预测
- **阶段**: 第四至第五阶段
